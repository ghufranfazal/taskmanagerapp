# Task Manager App

A simple browser-based task manager built with HTML, CSS, and JavaScript.

## Overview

This project provides a lightweight task management interface where users can add tasks, categorize them, mark tasks as completed, edit existing tasks, and delete tasks individually or in bulk.

## Features

- Add new tasks with a title and category
- View task cards with category labels
- Mark tasks as completed
- Edit task details after creation
- Delete tasks individually
- Clear all tasks with a single button

## Project Structure

- `index.html` — Main application layout and interface
- `style.css` — Responsive styling for the task manager
- `script.js` — Client-side behavior and task handling logic

## Usage

1. Open `index.html` in a web browser.
2. Enter a task description in the input field.
3. Choose a category from the dropdown.
4. Click the `Add` button to create the task.
5. Use the `Complete`, `Edit`, or `Delete` buttons on each task card.
6. Click `Delete All Task` to remove all tasks.

## Notes

- The app stores tasks only in memory for the current browser session.
- Task data is not persisted after page refresh.
